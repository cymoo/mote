# Mote 重设计 · 设计说明

> 原型入口：`Mote Redesign.html`（需经 HTTP 访问，如 `http://localhost:4311/mote-redesign/Mote%20Redesign.html`）
> 基准日期 2026-07-15；数据均为演示用模拟数据。

## 方向

**全新现代视觉，但保留 mote 的产品基因**：窄栏笔记流、热力图、标签树、三色标记、明暗双模式全部保留，
用更好的排版、层次、动效重新表达。关键词：纸感、克制、CJK 优先、举重若轻。

- **纸面层次**：底色带极淡的双角环境光晕（延续现有 `--theme-glow-*` 基因）；卡片用发丝级描边 + 分层软阴影，hover 轻浮起。
- **CJK 排版**：正文 15px / 行高 1.85 / 字距 0.012em；日期、统计数字、引用用衬线（Georgia/Songti SC）做点睛。
- **色彩克制**：每主题一个主色；标签、徽章一律用主色低透明度 tint；红绿蓝三色标记语义固定、跨主题不变。

## 主题（8 套精选为 4 套，明暗各一）

| 主题 | 性格 | 主色 | 圆角 | 对应替换 |
| --- | --- | --- | --- | --- |
| **青 celadon（默认）** | 青瓷苍绿 + 茶褐 | `hsl(170 58% 30%)` | 13px | moss / candy 合并 |
| 曦 dawn | 晨光暖纸 + 蔚蓝 | `hsl(216 78% 49%)` | 11px | default |
| 墨 ink | 无彩编辑部 + 朱砂 | `hsl(9 68% 46%)` | 6px | ink / blueprint 合并 |
| 紫 wisteria | 紫藤薄暮 + 玫瑰 | `hsl(278 56% 52%)` | 14px | nocturne / fjord / sakura / aurora 合并 |

令牌命名与 `frontend/src/index.css` 对齐（`--background/--foreground/--card/--muted-foreground/--border/--primary/…`），
新增：`--card-2`（悬浮/内嵌面）、`--faint`（第三级文字）、`--primary-tint(-2)`、`--shadow-1/2/3`、`--glow-a/b`。
落地时把每块值翻译回 HSL 三元组即可无缝接入现有 `[data-color-theme][data-theme]` 机制。

## 布局决策

**笔记页**：侧栏 282px + 内容 672px 居中。
- 侧栏：品牌点（主题色发光小圆点，随主题变化）+ 衬线统计数字（三列两端对齐铺满）+ 16 周热力图（两端对齐铺满侧栏、空格子可见、今日描边、hover 放大 + tooltip）+ 导航 pill 态 + 标签树（子级带引导线；悬停行尾出现操作菜单：置顶/重命名/删除）+ 底部设置/回收站。
- 笔记流：**按天分组**（衬线日期标签 + 发丝分隔线），卡片头部为时间/标记点/回应数/分享徽章，操作菜单 hover 才现身；正文标签为可点击 chip（点击即筛选）。
- 卡片菜单与现有 post-menu.tsx 功能一致：颜色标记行 → 编辑/引用/查看详情 → 导出 Markdown → 分享/删除 → 字数 · 更新时间脚注。
- 编辑器：聚焦时主色柔光描边；左侧插入工具、右侧三色标记 + 发布（**小屏隐藏三色标记**，颜色改由卡片菜单承担，避免工具栏拥挤）；发布后新卡片带高亮入场动画。

**搜索页**（2026-07 重做，替换原顶栏内联过滤）：对齐真实产品基因——从顶栏搜索入口滑出的**独立搜索面**、正文全文关键词、`相关度 / 最新 / 最早` 排序、结果即笔记卡片 + 内联 `<mark>` 命中高亮（主色 tint，跳过 #标签 chip）。
- 桌面：顶部下拉的居中浮层面板（发丝描边 + 分层阴影 + 背景模糊）；`⌘K` / `/` 打开、`Esc` 关闭、`↑↓` 选中结果、`Enter` 定位。
- 移动：全屏抽屉（搜索图标入口在顶栏），空态给「最近搜索 + 常用标签」，点标签直接跳转对应筛选。
- 空态不留白：最近搜索（本地存储，可清除）+ 常用标签（置顶优先，带计数）+ 一行语法提示。

**云盘页**：左导航 218px + 大画布卡片（bg card、大圆角、描边阴影），Linear 式分区。
- 导航：返回笔记、三个入口带计数、底部上传说明——动作按钮集中在顶栏不重复，侧栏为未来入口留位。
- 工具栏：面包屑 chip、全局搜索、排序菜单、列表/网格分段切换、新建/上传（唯一入口）。
- 列表：CSS grid 行（复选框 hover 现身、类型着色图标块、分享徽章、大小/时间 tabular）；文件夹恒置顶；
  **单击行 = 打开**（打开是高频），多选走复选框 / ⌘·Ctrl·Shift+单击 / 表头全选。
- 网格：图片/视频全出血缩略图，其余类型大图标块；卡片 hover 浮起。
- 多选 → 底部居中胶囊操作栏（弹簧入场）；拖拽 → 全区虚线高亮；上传 → 右下角上传坞（进度条模拟）；
  预览 → **全屏灯箱**（覆盖含侧栏的整个应用，左右切换、Esc 关闭）；分享 → 密码开关 + 有效期对话框（真实增删分享列表）。

**移动端**（容器查询 ≤720px + iPhone 外框预览）：
- 笔记：顶栏（菜单/品牌/**搜索**/明暗）+ 抽屉侧栏 + 底部居中 FAB + **重做的底部编辑器**。
- 移动编辑器（2026-07 重做）：浮层 sheet ↔ 全屏可切换；`#` 触发**标签联想条**（贴键盘上方，含「新建」项、↑↓/Enter/Tab/点选）；**草稿本地自动保存**、重开恢复；44px 触达工具栏 + 字数 + 主行动「发布」；正文 16px 防 iOS 缩放；颜色标记仍由卡片菜单承担（不入编辑器，与真实产品一致）。
- 云盘：顶栏收窄 + 分段标签页 + 列表两行式（名称 + 次行元信息）+ 右下上传 FAB + 悬浮多选栏。

## 动效规范

150–320ms、`ease-out` 或轻弹簧 `cubic-bezier(0.34,1.56,0.64,1)`；卡片入场 45ms 级联；
明暗/主题切换 320ms 颜色过渡；全体动效遵循 `prefers-reduced-motion`。

## 原型内可操作

发布/删除/恢复笔记、标签与颜色筛选、标签置顶/重命名/删除、搜索、热力图 tooltip、明暗与 4 主题切换、
云盘的进入文件夹/排序/双视图/多选删除（真实移入回收站）/新建文件夹/上传模拟/预览/创建与撤销分享、桌面—手机切换（左下「调节」）。

## 按天分组 × 虚拟列表（落地答疑）

分组展示**不需要**引入分组数据结构。列表仍是扁平的时间倒序数组（虚拟列表天然适配），
只在渲染第 i 项时比较 `dayOf(items[i]) !== dayOf(items[i-1])`，成立就在该卡片上方多渲染一条日期分隔行——
本原型正是这样实现的。react-virtuoso 用 `itemContent(index)` 里做同样比较即可，分页加载追加数据也不影响；
若想要吸顶日期头，可换 `GroupedVirtuoso`（由已加载数据算 `groupCounts`），成本也只是十几行。

## 已知局限（原型 ≠ 实现）

- 编辑器为纯文本 + `#标签` 识别，未实现 Slate 富文本；图片为渐变占位。
- 统计页、博客页不在本次范围；分享链接为演示数据。
- 移动端为 402px 外框模拟，真机需按安全区微调。
