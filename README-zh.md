# Mote

Mote 是一个简单的个人笔记本和博客。**当前分支是 Rust 版** —— 后端位于 [api-rs](./api-rs)，基于 Axum + SQLite + Redis 构建。

![screenshot](./samples/screenshot.png)

## 功能

* **富文本编辑器** — 支持 Markdown 快捷键，草稿自动保存
* **Hashtag 系统** — 在正文中输入 `#标签` 即可自动创建标签；支持多级标签（`父标签/子标签`）；可置顶、重命名和删除
* **图集** — 为 memo 附加多张图片，以网格方式展示
* **博客分享** — 将任意 memo 发布为公开博客，拥有独立的访问链接
* **全文检索** — 在所有 memo 中搜索，支持按相关度、最新或最早排序
* **Memo 关联** — 引用或回复已有的 memo，随时可解除关联
* **颜色标记** — 为 memo 标记红、绿、蓝三种颜色，便于快速分类
* **活跃热力图** — 类 GitHub 风格的日历热力图，直观呈现每日写作频率
* **统计面板** — 显示 memo 总数、标签数和活跃天数
* **回收站** — 软删除，最多保留 30 天，支持恢复或永久删除
* **文件** — 内置迷你云盘：多级目录、拖拽上传（支持分片续传，单文件最大 4 GB，可整个文件夹上传）、服务端去重（相同内容共享存储）、复制/副本、拖拽移动、右键菜单、收藏、存储用量展示、浏览器内预览（图片 / 视频【记忆进度、倍速、画中画】/ 音频 / PDF / 文本【代码语法高亮】）、文件夹与多选一键打包下载、可设密码与有效期的分享链接（文件和文件夹均可，文件夹分享页可浏览子目录）、独立回收站（30 天保留期）
* **深色 / 浅色主题**
* **移动端适配**
* **中英双语界面** — 可在设置中切换

## 版本

Mote 刻意做成多语言实现：共享同一套 React 前端和完全相同的 HTTP API，用多种语言各实现一份后端。每个后端各自占用一个长期分支：

| 版本 | 分支 | 后端 |
| --- | --- | --- |
| Go（主版本） | `main` | `api-go` |
| Rust | `feat/api-rs` | `api-rs` |
| Kotlin | `feat/api-kt` | `api-kt` |
| Python | `feat/api-py` | `api-py` |

当前分支是 **`feat/api-rs`**。后端构建与运行见 [api-rs/README.md](./api-rs/README.md)；[frontend/](./frontend) 前端为各版本共享。

## 部署

生产环境部署参见 [deploy/](./deploy)，提供 nginx + Docker Compose 方案，含 HTTPS 证书自动申请和数据备份。注意：部署脚本目前构建的是 Go 后端（`deploy/Dockerfile.api`、`deploy/compose.yml`），本版本需相应改为构建 `api-rs`。
